# VISTA Project

VISTA 是一个图像库与AI相关的应用，包括前端和后端组件。

## 项目结构

- `vista-backend/`: Node.js Express 后端服务器
- `vistapj/`: 前端 HTML/CSS/JavaScript 应用
- `bin/`: 工具和二进制文件（已忽略）
- `.ssh/`: SSH 密钥（已忽略）

## 安装和运行

### 后端

```bash
cd vista-backend
npm install
npm start
```

### 前端

打开 `vistapj/index.html` 在浏览器中。
